# Keyword Tracking Quick Start Guide

## üöÄ 5-Minute Setup

### Step 1: Set Up Google Search Console (5 minutes)

1. **Go to Google Search Console**
   - Visit: https://search.google.com/search-console
   - Sign in with your Google account

2. **Add Your Property**
   - Click "Add Property"
   - Enter: `https://tecxmate.com`
   - Choose verification method (HTML file, meta tag, or DNS)

3. **Verify Ownership**
   - Follow verification instructions
   - Wait for verification (usually instant)

4. **Submit Sitemap**
   - Go to: Sitemaps ‚Üí Add new sitemap
   - Enter: `sitemap.xml`
   - Click "Submit"
   - Wait 1-2 days for Google to index

### Step 2: Set Up Tracking Spreadsheet (2 minutes)

1. **Run Setup Script**
   ```bash
   npm run setup-keyword-tracking
   ```

2. **Open Generated File**
   - Open `keyword-tracking.csv` in Google Sheets
   - Or import to Excel

3. **Start Tracking**
   - Check rankings weekly
   - Update positions in spreadsheet
   - Track changes over time

### Step 3: Check Rankings (Weekly - 10 minutes)

1. **Google Search Console**
   - Go to: Performance ‚Üí Search Results
   - Check "Queries" tab
   - See your keyword rankings

2. **Manual Check (Optional)**
   - Search for keywords in Google
   - Check your position
   - Update spreadsheet

3. **Review Analytics**
   - Go to: Google Analytics
   - Check: Acquisition ‚Üí Organic Search
   - See: Traffic by country

## üí° Pro Tips

1. **Be Patient**: SEO takes time (2-3 months minimum)
2. **Focus on Quality**: Great content ranks better
3. **Track Consistently**: Weekly tracking is best
4. **Monitor Competitors**: See what they rank for
5. **Update Old Posts**: Refresh content to improve rankings
6. **Build Backlinks**: Quality backlinks help rankings
7. **Mobile First**: Mobile rankings matter most
8. **User Experience**: Good UX improves rankings

---

## üö® Common Mistakes to Avoid

1. **Don't Check Daily**: Rankings fluctuate daily - weekly is enough
2. **Don't Focus on Single Keyword**: Track multiple keywords
3. **Don't Ignore Long-Tail Keywords**: They often convert better
4. **Don't Forget Country-Specific**: Track by country
5. **Don't Neglect Analytics**: Combine Search Console + Analytics
